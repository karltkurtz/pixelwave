// THE IDEA

PiGarage.com is a Raspberry Pi project site. For this project (PixelWave), a real LED matrix is sitting in my house. Anyone on the internet can visit this page, claim the board, and draw something for me to find when I get home. It's a little window into a stranger's living room — a guestbook made of light.

// THE HARDWARE
  • 🖥️
    Raspberry Pi 4 Runs the web server and controls the LED matrix
  • 🖥️
    Raspberry Pi 4 (camera) Dedicated Pi that streams the live camera view
  • 💡
    16x16 LED Matrix 256 individually addressable RGB LEDs
  • 🔌
    Jumper Wires & Wago Connectors Wiring the Pi to the matrix
  • 📷
    Raspberry Pi HQ Camera (IMX477) 12.3MP Sony sensor — live stream of the board
  • 🔭
    16mm Telephoto Lens (CS-mount) Fixed focal length
